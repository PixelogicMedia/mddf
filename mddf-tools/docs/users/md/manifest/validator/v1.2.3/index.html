Release Notes and Executables

Release 1.2.3

Status: Pre-Release Candidate

Documentation:

  • Version 1.2.3 is compatible with the v1.2.1 User's Guide
  • Manifest Validator

Executable

  • Windows: ManifestValidator-1.2.3.exe
  • Mac OS-X: ManifestValidator-1.2.3.dmg or ManifestValidator-1.2.3.pkg
  • Linux: ManifestValidator-1.2.3.deb or ManifestValidator-1.2.3-1.x86_64.rpm

Source code is on the Movie Labs mddf GitHub repository . If you would like more information, please contact us.

Overview

This release of the desktop Manifest Validation tool has been upgraded to use v1.1.1 of the mddf-lib core software. There have been no changes to either the basic functionality of the Manifest Validation tool or its user interface.
